## tmux チートシート

> プレフィックスキー: `Ctrl + a` (以下 `prefix` と表記)

### 基本操作

| 操作          | コマンド         |
| :---------- | :----------- |
| ヘルプを表示      | `prefix + ?` |
| デタッチ（一時中断）  | `prefix + d` |
| 設定ファイル再読み込み | `prefix + r` |

### セッション操作

#### コマンドライン

| 操作                     | コマンド                 |
| :----------------------- | :----------------------- |
| 新規セッション作成       | `tmux`                   |
| 名前付きセッション作成   | `tmux new -s <名前>`     |
| セッション一覧           | `tmux ls`                |
| セッションにアタッチ     | `tmux a`                 |
| 特定セッションにアタッチ | `tmux a -t <名前>`       |
| セッションを終了         | `tmux kill-session -t <名前>` |

#### tmux内

| 操作                 | コマンド     |
| :------------------- | :----------- |
| セッション一覧と切替 | `prefix + s` |
| セッション名を変更   | `prefix + $` |

### ウィンドウ操作

| 操作               | コマンド     |
| :----------------- | :----------- |
| 新規ウィンドウ作成 | `prefix + c` |
| 次のウィンドウへ   | `prefix + n` |
| 前のウィンドウへ   | `prefix + p` |
| 番号でウィンドウ移動 | `prefix + 0-9` |
| ウィンドウ一覧から選択 | `prefix + w` |
| ウィンドウ名を変更 | `prefix + ,` |
| ウィンドウを閉じる | `prefix + &` |

### ペイン操作

| 操作                 | コマンド            |
| :------------------- | :------------------ |
| 左右に分割           | `prefix + \|`       |
| 上下に分割           | `prefix + -`        |
| 左のペインへ移動     | `prefix + h`        |
| 下のペインへ移動     | `prefix + j`        |
| 上のペインへ移動     | `prefix + k`        |
| 右のペインへ移動     | `prefix + l`        |
| 次のペインへ移動     | `prefix + o`        |
| ペインを矢印で移動   | `prefix + 矢印キー` |
| ペインサイズ変更     | `prefix + Ctrl + 矢印` |
| ペインをズーム/戻す  | `prefix + z`        |
| ペインを閉じる       | `prefix + x`        |

### コピーモード

| 操作               | コマンド               |
| :----------------- | :--------------------- |
| コピーモードに入る | `prefix + [`           |
| 移動               | `h, j, k, l` または矢印 |
| ページ上へ         | `Ctrl + u`             |
| ページ下へ         | `Ctrl + d`             |
| 検索（下方向）     | `/`                    |
| 検索（上方向）     | `?`                    |
| 次の検索結果       | `n`                    |
| 前の検索結果       | `N`                    |
| 選択開始           | `Space`                |
| コピー             | `Enter`                |
| コピーモード終了   | `q`                    |
| ペースト           | `prefix + ]`           |

### よく使うワークフロー

#### プロジェクト開始

```bash
# プロジェクト専用セッション作成
tmux new -s my-project

# ウィンドウに名前をつける
prefix + ,

# 新しいウィンドウを追加
prefix + c
```

#### 画面分割レイアウト

```bash
# 左右分割
prefix + |

# 右側をさらに上下分割
prefix + l  # 右へ移動
prefix + -  # 上下分割
```

#### 作業の中断と再開

```bash
# 中断（デタッチ）
prefix + d

# 再開（アタッチ）
tmux a -t my-project
```

### 便利な Tips

#### マウス操作

設定で `set -g mouse on` を有効にしているため：
- ペインをクリックで選択
- ペインの境界をドラッグしてサイズ変更
- マウスホイールでスクロール（自動的にコピーモードに入る）

#### ペインのズーム

`prefix + z` で現在のペインを一時的に全画面表示できます。
もう一度 `prefix + z` で元に戻ります。

### 設定ファイルの場所

- **tmux設定**: `~/.tmux.conf`
- **設定反映**: `prefix + r` または `tmux source-file ~/.tmux.conf`


#### 色が正しく表示されない

`~/.tmux.conf` に以下が設定されているか確認：
```bash
set -g default-terminal "screen-256color"
```

#### コピーモードの選択がうまくいかない

- `Space` で選択開始
- 矢印キーまたは `h,j,k,l` で範囲選択
- `Enter` でコピー
- コピーモードを抜けてから `prefix + ]` でペースト
